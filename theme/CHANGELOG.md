# Changelog

All notable changes to the K3S Admin Theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-11

### Added
- Initial release of K3S Admin Theme
- Modern gradient-based color palette (Primary, Success, Warning, Danger, Info)
- Complete SCSS variable system with CSS custom properties
- Comprehensive mixin library for gradients, components, and responsive design
- Pre-built component styles (cards, buttons, badges, inputs)
- Full Angular Material integration with custom palettes
- Utility class system for rapid development
- Dark mode support (system preference + manual toggle)
- Responsive design mixins and breakpoints
- Glassmorphism effects
- Custom scrollbar styling
- Fade-in and slide-in animations
- Hover effects (lift, scale)
- Grid and flexbox utilities
- Typography utilities
- Spacing utilities (margin, padding)
- Shadow utilities (including colored shadows)
- Border radius utilities
- Complete documentation with examples
- MIT License
